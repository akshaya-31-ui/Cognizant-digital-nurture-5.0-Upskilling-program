## Project Overview
This repository contains a unified showcase of engineering competencies requested during the Cognizant Digital Nurture 5.0 upskilling journey. Over a rigorous three-week period, all milestone tasks spanning client-side presentation layers, relational databases, and core algorithmic program structures were implemented, tested locally, and compiled into this structured workspace.

The repository is organized directly around the recommended learning sequence tracks:
1. **Module 1 (Frontend UI & Logic):** Local Community Event Portal (HTML5, CSS3, Bootstrap 5, Vanilla JavaScript, and jQuery integration).
2. **Module 2 (Database Layer):** 25 ANSI SQL relational architecture queries.
3. **Module 3 (Core Backend Engine):** 41 Object-Oriented Java programmatic assignments.

## Key Implementation Highlights

### Module 1: Frontend Web Development
* **Two-Tier Navigation Gate:** Secure, smooth transition from a floating-label Login/Signup tab component to the core dashboard.
* **Smart Session Banner:** Uses `sessionStorage` and `localStorage` to identify returning profiles and dynamically display a "Welcome back user" banner.
* **Semantic UI Layout:** Clean structure utilizing HTML5 tags, Bootstrap 5 responsive grids, and a newspaper-style multi-column newsletter section.
* **Interactive Event Handlers:** Real-time form validations, text character counters, double-click image zoom scaling, and dynamic closure category counters.
* **Async Mock APIs:** Simulates server communication using JS Promises and `async/await` paired with user loading spinners and error-catching validation blocks.
* **Advanced Web APIs:** Localized button block utilizing Web Geolocation coordinates alongside cache-clearing memory mechanics.

### Module 2: ANSI SQL Relational Queries
* **25 Production Queries:** Validated inside MySQL Workbench tracking complete data operations.
* **Database Commands:** Comprehensive coverage of DDL/DML scripts, multi-table JOINs, primary/foreign key relations, and custom query constraints.

### Module 3: Core Java Programming
* **41 Independent Labs:** Algorithmic backend source code written and validated inside VS Code.
* **OOP Paradigms:** Strong foundational application of class structures, encapsulation properties, data array loops, and try-catch exception matrices.
